# MakeUs 6기 데모데이 최우수상
## 프로젝트 간략소개
* 둥지메이트는 할일/메모/공지/투표와 같은 기능으로 같은 공간을 사용하는 메이트 간 
집안일을 공유하고 분담할 수 있는 서비스입니다.
* 2021년 3월부터 4월까지 안드로이드 개발자분들과 협업해서 개발을 진행했고 2021년 6월까지 서비스를 운영했습니다.
이후 혼자서 백엔드 리팩토링을 진행했습니다.
